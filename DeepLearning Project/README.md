This project focuses on modeling the progression of diabetes using the Diabetes dataset from sklearn by applying an Artificial Neural Network (ANN). The dataset is first preprocessed through normalization and analyzed using exploratory data analysis to understand feature relationships. An ANN model is then built, trained, and evaluated using metrics like Mean Squared Error and R² score. Further improvements are made by experimenting with different architectures, activation functions, and hyperparameters such as learning rate and dropout. The final model provides insights into how various factors influence diabetes progression and demonstrates the effectiveness of ANN in predictive healthcare analysis.
